Simple REST API Authentication plugin for WordPress - a powerful solution for integrating your website with external applications. 
This plugin allows you to use WooCommerce REST API credentials for WordPress REST API and offers debugging capabilities for troubleshooting. It also provides the flexibility to enable or disable the SSL requirement for added security.

== Description ==

Introducing our **Simple REST API Authentication** plugin for WordPress - the ultimate solution for seamless integration between your website and external applications. 

This plugin lets you use Basic Authentication for WordPress REST API and other custom-developed endpoints as well as third-party plugin REST API endpoints with **WooCommerce REST API credentials**. 

With Simple REST API Authentication plugin, you won't have to use different credentials or expose your user name and password to upload media to WordPress and then use it for WooCommerce products.

= Features =
* Seamless integration with WooCommerce REST API credentials for WordPress REST API
* Debugging capabilities for troubleshooting
* Flexibility to enable or disable the SSL requirement for added security.

The WordPress REST API is a powerful tool that makes CRUD (Create, Read, Update & Delete) operations available from anywhere, instead of being limited to the admin dashboard. 

It provides a lightweight form of communication between the client and the server, making it a great solution for exchanging data. 

This plugin can be used to create iOS/Android, etc native apps, and we can use any language we want as long as the language can make HTTP requests and interpret JSON such as Node.js, Express.js, Ruby, Python etc.

The possibilities are endless with this plugin, and it can be used in hundreds of various scenarios to make your website more accessible, efficient and user-friendly.

== Installation ==
1. Go to Wordpress -> Plugins -> Add New
2. Hit **Upload Plugin** button
3. Hit **Choose File** button and select zip file with the plugin
4. Hit **Install Now** button
5. Hit **Activate** link
6. Installation complete
